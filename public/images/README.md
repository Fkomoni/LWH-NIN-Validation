# public/images/

Drop site-wide photographic assets here. The landing page automatically
picks up anything at the paths below.

## `hero-family.jpg`

Shown on the right-hand side of the landing-page hero. Replace the file
(or upload a same-named file via Render's deploy) and the image appears
on the next request.

- **Aspect ratio**: 4:3 (landscape) — the container enforces this, so
  upload at that ratio to avoid cropping.
- **Recommended size**: 1200×900 (or larger; browsers will scale down).
- **Format**: JPG or PNG. JPG renders faster at this size.
- **Subject**: the brand tone manual shows warm, benefit-led imagery —
  campaigns use smiling people, families, older couples, children in
  healthcare contexts. Any of those works.
- **If the file is missing**: the page silently falls back to a
  branded gradient with subtle dot pattern and the "For health, wealth
  & more" overlay — it's designed to look intentional, not broken.

## Other images

Feel free to add more (`hero-success.jpg`, `admin-cover.jpg`, etc.) and
wire them into the corresponding components. Keep each under ~300 KB
so first-load stays fast.
